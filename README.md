# electron_SelectEatMenu

一个基于 **Electron + Vue3 + TypeScript + Vite** 的桌面应用项目，用于解决“每天吃什么”的选择困难。

## 功能目标
- 支持动态导入菜单配置
- 每日随机选择菜单项
- 基于历史记录的“冷却”机制：一段时间内避免重复推荐近期吃过的内容

## 技术栈
- Electron
- Vue 3
- TypeScript
- Vite

## 开发
```bash
npm install
npm run dev
```

## 构建
```bash
npm run build
```

## OpenSpec
本项目使用 OpenSpec 进行规格驱动开发（spec-driven development）。
